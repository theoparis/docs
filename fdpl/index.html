Installation git clone https://github.com/creepinson/fdpl cd fdpl Getting Started The configuration file for fdpl is made with typescript and is located at fdpl.ts in the root of the project. Configuration Example import type { InputConfig } from "@flowtr/fdpl"; const config: InputConfig = { database: { type: "sqlite", uri: `sqlite://${process.cwd()}/data/fdpl.sqlite`, }, apps: [ { name: "web", build: { image: "nginx:alpine", }, expose: [ { port: { from: 80, }, host: "test.localhost", }, ], }, ], }; export default config; Configuration Breakdown The first section in the configuration specifies the location of the database.
